## 📥 Cara Mendapatkan Dataset Lengkap

Karena keterbatasan ukuran repositori GitHub, dataset lengkap (3.399 sequence) tidak disertakan langsung di repositori ini. 

Anda dapat mengunduh dataset lengkap melalui tautan berikut:
- 📂 **Dataset Lengkap (.npy)**: [(https://drive.google.com/file/d/1pmZ8SWNa_1PHjufIOMkrrz4APNGCz8uU/view?usp=sharing)]

Setelah diunduh, ekstrak file tersebut dan letakkan di dalam folder `dataset/`sesuai struktur direktori di bawah ini.

dataset/pukul

dataset/redam

dataset/netral
